ToApp

Convert any web page into mobile app. To download the app, simply click add to homescreen from your web browser.

For Developers

Simply paste the following code in your header to make your website a mobile app (Progressive Web App).

<script src="https://realmichaelye.github.io/ToApp/toapp.js"></script>
